Readme.txt
-----------------

Main execution file :
final_project_main.py

Data files required to be in the same folder :
adminpwd.txt
cust_profile.csv
foodmaster.csv
cust_ord_history.csv

Login :

- Admin login uses the password stored in adminpwd.txt
- Customer logs in using a 10 digit mobile number and password as stored in cust_profile.csv

The Administrator does not have an option to delete an item from the menu. 
The Administartor can use 'available on the menu' option to withdraw items from the menu or bring them back. 

The menu items are being retained to maintain data integrity with the customer's order history
